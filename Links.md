## 💡 How to use these in your HTML

To use any of these as a favicon, simply copy the **CDN Link** and paste it into a standard HTML `<link>` tag in your `<head>`:

```html
<!-- Example: Setting the main Axinity Logo as the Favicon -->
<link rel="icon" href="https://cdn.jsdelivr.net/gh/ETERNALKIRITO/AxinityIcons@main/Icons/Axinity-Logo.png" />

<!-- Example: Setting the 180px Developer Logo for Apple Devices -->
<link rel="apple-touch-icon" href="https://cdn.jsdelivr.net/gh/ETERNALKIRITO/AxinityIcons@main/Icons/Variations/Axinity-Developer-Logos/PNGs/Axin-Dev-Logo_180.png" />
```
